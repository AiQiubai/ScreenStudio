# ScreenStudio 优势

- 自动加平滑缩放、光标动画，直接出精美成片。很适合做教程视频、产品宣传片；
- 👉 如果你是 Mac 用户 + 想要高质感成片 → Screen Studio 是首选。
- 软件自带吸引眼球的效果，手势鼠标；
- ❗Screenstudio 是 Mac-Only 软件，无Windows版本，无安卓版本。

# Screenstudio、QuickTime与OBS对比

| 功能 | 核心卖点 | 后期工作 | 适用人群 | 价格 | 平台 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| Screen Studio | 自动平滑缩放与运镜 | **几乎为零** | 高质量教程、产品演示 | 付费 (较高) | 仅 Mac |
| QuickTime (Mac自带) | 免费、简单 | 巨大 (需手动剪辑) | 个人简单录制 | 免费 | 仅 Mac |
| OBS | 免费、强大、可直播 | 巨大 (需手动剪辑) | 游戏直播、复杂串流 | 免费 | 全平台 |
| Loom / Tella.tv | 快速分享、带摄像头画中画 | 较少 (但无自动缩放) | 团队协作、快速反馈 | 免费版/订阅制 | 全平台 (Web/桌面) |

结论：

如果你要直播，用 OBS。

如果你要快速给同事发个反馈，用 Loom。

如果你要制作一个公开发布的、精美的产品教程或营销视频，Screen Studio 是唯一选。
